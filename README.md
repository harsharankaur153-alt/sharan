# OpenEnv Workplace Simulation

## Overview
This project simulates real-world workplace tasks such as email handling, scheduling, and classification.

## Tasks
1. Easy: Email classification
2. Medium: Email reply generation
3. Hard: Meeting scheduling

## Features
- OpenEnv compliant environment
- Step-based reward system
- Deterministic grading
- Baseline inference script

## Run

pip install -r requirements.txt
python inference.py

## Output Format
[START]
[STEP]
[END]
